# mote-ros

ROS2 drivers for Mote.

## Using Pixi

Install...
* [Pixi](https://pixi.prefix.dev/latest/installation/#__tabbed_1_1)
* [Just](https://just.systems/man/en/introduction.html)

```
just install
just shell
just build
just test
```

## Using System Jazzy Installation

Install...
* [Just](https://just.systems/man/en/introduction.html)

```
just install-system
just build-system
just test-system
```
